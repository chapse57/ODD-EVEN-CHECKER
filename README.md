## 📸 이런 화면이 나와요!

> 어두운 배경에 격자 무늬가 깔린 세련된 디자인의 홀짝 판별기입니다.

- 상단에 **"홀짝 판별기"** 제목과 **"ODD & EVEN CHECKER"** 부제목이 표시됩니다
- 가운데 입력창에 숫자를 입력하세요
- 파란색 **"판별하기"** 버튼을 누르면 결과가 나타납니다
- 짝수면 **파란색**, 홀수면 **빨간색**으로 결과가 표시됩니다
- 최근에 입력한 숫자 기록도 하단에 자동으로 저장됩니다

---

## 🚀 사용 방법

1. 위 링크로 접속합니다
2. 입력창에 **정수**를 입력합니다
3. **"판별하기"** 버튼을 클릭하거나 **Enter** 키를 누릅니다
4. 결과를 확인합니다! ✅
